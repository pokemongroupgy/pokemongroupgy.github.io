Pokemon Masters: Tier List And Sync Pair Ranking

February 1, 2020 Gaurav Yadav

Pokemon Masters trainers have a specific partner Pomeon known as Sync pairs. These Sync pairs have different abilities and usefulness in battle, so we have divided them into tiers and rankings.

In Pokemon Masters tier list and ranking we will focus on finding best Sync pairs according to their roles.

Table of Contents

  • Strike (Special) Tier List
  • Strike (Physical) Tier List
  • Tech Tier List

Tier List Analysis

This Tier List is based on Tier Points. Tier Points = Stat Points + Move Effect Points + Adjustment Points Stat Points:
  • Support: Stat Points = Bulk (HP/2.75 + Sp.Def + Def) / 30.
  • Strike (Physical): Stat Points = Attack / 10.
  • Strike (Special): Stat Points = Sp. Atk / 10.
  • Tech: Stat Points = Stat total across all six stats / 60.
Move Effect Points:
  • One point for each effect for a SYnc Pair [ ATK Up, Sp ATK Up, SPD Up, Crit Up, Move Gauge Up, Various Stats Up, Dual Type, AoE Damage, AoE Immunity ]
  • 3x bonus if they target all allies.
  • 2x bonus if the stat bonus “Sharply raises” the corresponding stat.
  • 3x bonus if the stat bonus “Drastically raises” the corresponding stat.
Adjustment Points:
  • Skyla & Swanna (+5 points) – higher placement due to the higher value of her full party Speed boost and team synergy.
A.I. targets the Pokemon with max Bulk, so Bulk is also a factor. Usually, a Support that has high bulk will protect other more fragile Strike-type (damage dealer) Sync Pairs from harm.
