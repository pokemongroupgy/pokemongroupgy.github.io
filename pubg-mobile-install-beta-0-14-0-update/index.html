PUBG Mobile: How To Install Beta 0.14.0 Update

August 7, 2019 Gaurav Yadav

PUBG Mobile beta version 0.14.0 update is now officially released and the update brings several changes.  Some of the changes are a new Zombie Infected mode, Daily mission improvements, new items/cosmetics, UI changes, and several bug fixes.

The final version of the update has not been released, but players can try new features by downloading the beta version of the game for both Android and iOS.

Installing Beta

First, download the installation file for Android/ iOS from the links given below.

Android PUBG Mobile Beta:

https://filecdn-igamecj.akamaized.net/fclient/download.html

iOS PUBG Mobile Beta:

https://testflight.apple.com/join/yfCvLkdl

Steps for Android installation:

After downloading the APK file from the above link, find the file named Android_IG0140_No4_0.14.0.11190_Shipping_Google_CE.signed.shell.apk in download manager and click on it to install.

You need to allow installation from unknown sources in your settings. After the installation is complete (it might take some time based on your device performance) open the game and use Guest login to play the Beta version.

Steps for iOS installation:

After downloading the file from the above link, install the TestFlight app on your device. Click on the Beta download link again and select Start Testing option in TestFlight, Hit Install/Update button to install the game.
